Requirements

Why we have requirements

We have requirements to who can use Mattcoin because we need to ensure the integrity of Mattcoin and find its target audience. Theres really only a few and well explain all of them.

You need to have a Scratcher Status Account

Mattcoin can be embedded in projects so people can donate to other people or/and buy ingame items withing scratch projects. All this requires some mild use of cloud data. We also require Scratcher status because is how us, Mattcoin, makes sure that people don't attempt to game the system with bots/

A few more stuff

For all support please vist: https://scratch.mit.edu/users/mattcoinsupport. We also have a github repository (https://github.com/mtcscratch) where you can find the nodejs miner to participate in the Mattcoin block chain.
